### React-Redux simple boilerplate
> 没那么复杂的 react-redux 脚手架。

### Usage
> 1、git clone  
> 2、安装依赖：$ npm install  
> 3、启动服务：$ npm start  
> 4、生成文件：$ npm run build  

### Features
* 开发环境单文件打包
* 生产环境分块打包
* Less代替Css
* 内置 react-redux demo


### 依赖
* UI框架：React & React-Dom
* 路由：React-Router
* 状态管理：React-Redux
* JS：ES6 (添加 ES7 部分支持)
* 样式：Less/Css
* 网络请求：Fetch
* 打包构建：Webpack2 Babel
* 包管理：Npm

### webpack2
* 打包 css (extract-text-webpack-plugin)
* 自动处理浏览器前缀 (autoprefixer)
* 自动生成 html (html-webpack-plugin)
* 打包前清除上次文件 (clean-webpack-plugin)
